# 导航页制作工具 (NavBuilder)

这是一个基于 React 和 Tailwind CSS 开发的便捷导航页面生成工具。用户可以通过简单的界面输入链接信息，实时预览效果，并一键导出包含 HTML 和 CSS 的 ZIP 压缩包。

## 主要功能

- **可视化编辑**：支持自定义站点标题，添加/删除导航链接。
- **详尽信息**：每个链接可配置标题、URL、描述说明及 LOGO 图标。
- **智能图标**：
  - 支持自定义图片 URL 作为 LOGO。
  - 未配置 LOGO 时，自动显示标题首字母。
  - 若标题也为空，则显示默认的地球图标。
- **实时预览**：内置仿真浏览器窗口，所见即所得。
- **一键导出**：自动打包 `index.html`、`style.css`、`Inter-latin.woff2` 和 `README.md` 为 ZIP 文件，生成的结果完全自包含，不依赖外部字体库。
- **配置备份**：支持导出和导入 JSON 格式的配置文件，方便随时恢复或迁移编辑进度。
- **多端适配**：导出的页面支持部署在域名的根目录或任意子目录下。
- **响应式设计**：生成的导航页完美适配手机、平板和电脑桌面。
- **精美视觉**：采用 #BFA5FF 作为主色调，现代化的卡片式 UI 设计。

## 技术栈

- **前端框架**：React 19
- **样式处理**：Tailwind CSS 4
- **动画效果**：Motion (framer-motion)
- **图标库**：Lucide React
- **打包工具**：JSZip & FileSaver.js

## 如何使用

1. **输入信息**：在左侧面板设置您的站点标题，并添加目标页面链接。
2. **实时预览**：在右侧预览区域查看生成的导航页效果。
3. **备份配置**：点击“导出配置”下载 JSON 文件，下次使用时通过“导入配置”即可快速恢复。
4. **生成发布**：点击右上角的“导出 ZIP”按钮，解压后将其中的文件上传到任何静态网站托管服务即可。

## 配置文件格式 (JSON)

导出的配置文件采用 JSON 格式，结构如下：

```json
{
  "siteTitle": "我的导航页",
  "items": [
    {
      "id": "随机ID",
      "title": "页面标题",
      "url": "https://...",
      "description": "简短描述",
      "logo": "图标URL (可选)"
    }
  ]
}
```

## 开发与运行

如果您想在本地运行此项目：

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 导出与部署

在本地运行或预览后，您可以使用界面右上角的"导出 ZIP"功能：
1. 下载生成的 `.zip` 文件。
2. 解压文件，您会看到 `index.html`，`style.css`，`Inter-latin.woff2` 和 `README.md`。
3. 这些文件是纯静态文件，可以直接双击 `index.html` 在浏览器中预览。
4. 将解压后的所有文件上传到静态托管服务（如 GitHub Pages, Vercel, Netlify 或您自己的服务器）即可发布。由于资源引用已改为相对路径，该页面可以部署在任意子目录下。
